== Description ==

This plugin will help you embed public fb videos on your wordpress site using a shortcode with ease. Use the shortcode [ae-fb-embed url="FB_VIDEO_URL"] and your done.
You can also use the video width attribute like [ae-fb-embed url="FB_VIDEO_URL" width="800"] to specify your desired video width. Enjoy!

== Installation ==

1. Upload `ae-fb-video-embed` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
